# python-warmup

Small standard-library-only Python utilities built while learning Python,
as part of a data-engineering learning journey. No external runtime
dependencies — the goal is core-language fluency.

## Utilities

| Script | What it does |
|---|---|
| `csv_transformer.py` | Reads a CSV, cleans it (whitespace, type coercion, missing values), adds derived columns, writes a cleaned CSV. |
| `word_counter.py` | Reads a text file and outputs the top-N most frequent words, with deterministic tie-breaking. |
| `log_parser.py` | Generates synthetic logs, parses them into typed records, and summarizes by level, busiest hour, and top messages. |

## Setup

Requires Python 3.12+ and [Poetry](https://python-poetry.org/).

```bash
poetry install
```

## Usage

```bash
poetry run python csv_transformer.py     # writes titanic_cleaned.csv
poetry run python word_counter.py        # writes pride_word_counted.csv
poetry run python log_parser.py          # writes log.txt, prints a summary
```

(Data files: `titanic.csv` from the DataScienceDojo dataset; `pride.txt`
from Project Gutenberg. `log.txt` is generated by the log utility.)

## Testing & linting

```bash
poetry run pytest -v        # run all tests
poetry run ruff check .     # lint
```
